# Chape Expert - Site de Vente de Chapes de Béton

Un site web moderne et responsive pour une entreprise spécialisée dans l'installation de chapes de béton.

## 🚀 Fonctionnalités

- **Design moderne** avec interface utilisateur intuitive
- **Responsive design** optimisé mobile-first
- **Navigation fluide** avec défilement automatique
- **Sections complètes** : Accueil, Services, Réalisations, Contact
- **Formulaire de contact** fonctionnel
- **Support multilingue** (français)
- **Police Roboto** pour une typographie professionnelle

## 🏗️ Structure du Site

### Header

- Logo de l'entreprise
- Navigation principale avec liens d'ancrage
- Bouton d'appel téléphonique rapide
- Menu hamburger pour mobile

### Section Hero

- Titre principal accrocheur
- Description des services
- Boutons d'action (appel téléphonique, services)

### Section Services

- 4 types de chapes : traditionnelle, anhydrite, auto-nivelante, isolante
- Cartes avec icônes et descriptions
- Design en grille responsive

### Section Réalisations

- Galerie de projets avec placeholders
- Informations sur chaque réalisation
- Badges pour les métrages et types de chapes

### Section Contact

- Formulaire de contact complet
- Informations de l'entreprise
- Cartes colorées pour chaque moyen de contact

### Footer

- Liens de navigation
- Informations de copyright
- Design épuré et professionnel

## 🛠️ Technologies Utilisées

- **Vue.js 3** avec Composition API
- **UnoCSS** pour le styling
- **Vue I18n** pour l'internationalisation
- **Vite** pour le build et le développement
- **TypeScript** pour la sécurité des types

## 📱 Design Mobile-First

Le site est conçu avec une approche mobile-first :

- Grilles responsive qui s'adaptent à tous les écrans
- Navigation optimisée pour mobile
- Boutons et formulaires adaptés au tactile
- Typographie lisible sur tous les appareils

## 🚀 Installation et Développement

```bash
# Installer les dépendances
pnpm install

# Lancer le serveur de développement
pnpm dev

# Build pour la production
pnpm build

# Prévisualiser la production
pnpm preview
```

## 📝 Personnalisation

### Ajouter des Réalisations

1. Modifier le composant `Projects.vue`
2. Remplacer les placeholders par de vraies images
3. Mettre à jour les descriptions et métrages

### Modifier les Services

1. Éditer le fichier `locales/fr.yml`
2. Ajuster les descriptions des types de chapes
3. Personnaliser les icônes SVG

### Changer les Informations de Contact

1. Mettre à jour le composant `Contact.vue`
2. Modifier les numéros de téléphone et emails
3. Ajuster l'adresse et les horaires
